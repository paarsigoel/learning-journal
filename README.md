### Day 2 — June 2026

**What I learned:** The 5-part prompt formula — Role, Context, Task, Constraints, Output Format.
Every powerful prompt uses this structure. Without it, AI gives generic answers.
With it, AI gives expert-level, tailored responses.

**What I built:** My first 5-part prompt — a Physics study guide on Vectors for JEE Foundation.
Also set up my GitHub profile README and pinned my first repos.

**Key insight:** The AI doesn't get smarter when you write a better prompt.
YOU get smarter about how to use it. The formula is the skill.

**What I practised:** Writing a prompt for my Vectors test using all 5 parts.
Got feedback that my Context and Constraints were strong.
Need to avoid asking for images/diagrams since AI can't generate those inline.
